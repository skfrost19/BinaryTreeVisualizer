A simple binary tree visualiser made with express-js and vis-js.

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Usage

-   Enter the nodes
-   Enter the edges
-   Click on the button to visualise the tree
-   Click on reset button to reset the board
